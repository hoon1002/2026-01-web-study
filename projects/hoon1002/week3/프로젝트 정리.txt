1. 프로젝트 소개

제 프로젝트는 새로운 취미를 시작하는 사람들이 시작하는 방법을 쉽게 찾기 위해 정보를 주고받는 커뮤니티 기능을 하는 웹 플랫폼입니다.

주요 기능
 - 가이드 탐색 및 검색 : 카테고리별, 태그별 분류 및 키워드 검색 기능
 - 로그인 및 세션 관리 : 로그인을 통한 계정 관리
 - 고인물의 가이드 작성
 - 인터랙션 : 마음에 드는 가이드에 '좋아요' 또는 '스크랩' 기능
 - 마이페이지 : 내가 작성한 글, 좋아요한 글, 스크랩한 글 볼 수 있음

2. 프로젝트 파일 구성 및 역할

지금까지 내용은 순수 HTML, CSS, JavaScript로 구성

HTML
 - index.html : 메인 화면, 메인 텍스트, 검색창, 태그, 카테고리 등등을 띄워놓음
 - category.html : 카테고리/태그/검색어에 따른 전체 가이드 목록을 필터링하여 보여주는 페이지
 - guide.html : 개별 가이드의 상세 내용을 읽고, 좋아요/스크랩을 누를 수 있는 페이지
 - write.html : 새로운 가이드를 작성하기 위한 입력 폼 페이지
 - mypage.html : 사용자의 활동 내역(스크랩 등)을 모아보는 프로필 페이지

CSS
 - css/style.css : 웹 사이트 전체의 디자인 규칙을 정의하는 css파일, 페이지의 색상, 반응형 레이아웃, 버튼 디자인, 애니메이션 효과 등

JavaScript
 - js/data.js : 지금은 서버가 없으므로 초기 데이터를 정의해주는 스크립트(카테고리 목록, 태그, 등등)
 - js/app.js : 웹 전반에서 공통으로 사용되는 핵심 로직(로그인, 데이터 저장/불러오기, 상단 바, 맨위로가기 버튼 등)
 - js/home.js, category.js, guide.js, write.js, mypage.js : 각각 html과 대응되는 화면 렌더링과 이벤트를 처리하는 스크립트 파일

3. 연결구조
css파일은 <link rel="stylesheet" href="css/style.css"> 태그를 html 파일에 작성하여 html 파일의 스타일을 변경

js파일은 <script> 태그를 통해 페이지의 로직을 작동
